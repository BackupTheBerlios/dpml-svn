Products

Avalon consists of a number of product and sub-projects.

Name Description
Framework

The framework collection includes a component meta model used to describe component dependencies towards a container, and, the classic framework APIs that describe the runtime contact between a container and a component.
Merlin

A container is a platform supporting the management of components. This page provides links to Merlin Avalon's reference implementation together with general containment theory, and references to legacy products.

Planet

The components page provides a summary of the two main component groups in Avalon - the course grained cornerstone suite and the fine grain utilites under Excalibur, together with general strategies concerning component repository management.
