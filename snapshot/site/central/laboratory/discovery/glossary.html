Discovery Terminology
Glossary
TermDescription
Identifiable An object supporting URI based identification.
Artifact A reference to a repository based resource.
ResourceDescription A type describing an Internet based resource.
Disclosure A type describing an act of disclosure of an Artifact.
Receipt A type used to provide proof to a client of an act dealing with registration, modification, or retraction of an artifact from a Registry.
Registry An service through which a client can register a resource description or existing disclosure instance.
Directory An interface through which a client can invoke requests enabling the discovery of artifact descriptions based on supplied filters.
