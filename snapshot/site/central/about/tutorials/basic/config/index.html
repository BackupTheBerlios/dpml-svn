TopicSummary
Using Tutorial describes the Configurable lifecycle interface and how this is used by a component implementation to receive a configuration instance.
Defaults Tutorial covering the creation of a default configuration and association of the default with a particular component type.
Declaring This tutorial covers the declaration of a configuration as part of the <component> tag within a block directive.
Overriding This tutorial covers the declaration of a set of configuration targets that override configuration declarations in a block.xml directive.

Resource supporting this tutorial are contained in the turorials/configuration package.
