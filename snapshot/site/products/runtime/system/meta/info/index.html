TopicSummary
Type Definition of the component type. A Type meta info object describes the structural features of component at the level equivalent to a class. A type includes information about deployment and runtime dependencies together with arbitrary attributes, a component implementation version and a component type name.
Service A service is a meta info object that describes a service provided by a component type. Service supports the attribution of a version and arbitrary properties.
Attributes Attribute declarations used in conjunction with meta-info descriptors supporting customisation of the meta-info model.
