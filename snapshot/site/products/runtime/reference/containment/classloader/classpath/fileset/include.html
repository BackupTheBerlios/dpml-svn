Include Declarations
Description

Declaration of jar files to include within the scope of a fileset.

Attributes
AttributeRequiredDescription
nameyes The name of the file to include. Ant-like wildcard patterns are supported.
Sample XML This example includes all the jar files in the lib directory and in all its subdirectories. 
<fileset dir="lib">
  <include name="**/*.jar"/>
</fileset>
