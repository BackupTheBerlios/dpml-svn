Parameters Directive
Description

A parameters directive is used by Merlin to construct a Parameters argument when servicing a Parameterizable object.

Nested Elements
ElementCardinalityDescription
parametern Declaration of the paramets to include within a Parameters instance.
Sample XML 
  <profile>
    <parameters>
      <parameter name="name" value="Gloria">
      <parameter name="size" value="142">
      <parameter name="color" value="red">
    </parameters>
  </profile>
