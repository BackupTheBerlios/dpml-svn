Legacy Content

Several Apache Avalon products have been either retired or spun off into their own Apache projects. For a more detailed record of Avalon's evolution, see the history section.

Product Description Current Status
Excalibur Excalibur included a component library as well as the Excalibur Component Manager (ECM) -- a minimal Avalon container. Graduated to it's own ASF Project: Apache Excalibur
Fortress Fortress is a lightweight Avalon container designed to to be the successor to ECM. Now part of the Apache Excalibur Project
Phoenix Avalon Phoenix is an Avalon 4 container designed to be used as an application server framework. Active development halted. See distribution mirrors for prior releases and documentation. Also, see Codehaus' Loom project, a fork of Phoenix.
