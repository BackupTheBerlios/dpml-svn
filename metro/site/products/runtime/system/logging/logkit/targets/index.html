Avalon LogKit Plugin
Overview

The LogKit plugin defines an local API for LogKit related log target factories and log targets. Through this API the LogKit implementation manages a set of standard targets and a target plugin framework.

Standard LogTarget
FactoryDescription
Stream The <stream> log target provides support for the logging of message to a console via System.out or System.err. If no logging targets are defined, the implementation will establish a default console target.
File The rotating <file> target provides support for a set of rotation strategies including rotation by size, rotation on data, and combined date/size rotation scenarios.
Multicast The <multicast> target is a target that handles the redirection of a logging event to one or more log targets.
Plugin The plugin <target> provides support for plugable log target factories.
